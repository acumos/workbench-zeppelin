Zeppelin Notebook Release Notes
===============================

Release 1.0, 18 Sep 2018
------------------------

   • Docker image build with Apache zeppelin 0.8.0
   • Integration with Acumos Model Manager and Data Source Manager.
   • Comes with a list of Machine Learning notebook templates.
   • Pre-configured interpreters for Livy, File, JDBC and Python.
